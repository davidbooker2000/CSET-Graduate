New University-wide Information Security Program

MOAT - A Requirement For All Users Of NSU Technology

As required by the Commonwealth of Virginia, all users of NSU technology are required to be aware of the policies which are part of NSU’s Security Program (ITRM Security Standard SEC501).Therefore, please go to https://va.moatusers.com, and read and accept the required policy agreement document before July 1, 2011.

 

A short video, which explains how to logon to MOAT, reset your password if needed, and accept the Security Program Agreement, is available at http://www.nsu.edu/itsecurity . Feel free to contact us at security@nsu.edu if you have any questions.
